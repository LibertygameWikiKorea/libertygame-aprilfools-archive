ChatGPT, Gemini, GitHub Copilot 등의 AI 챗봇을 모방하였습니다. 실제 존재하는 게임 제목을 요청한 경우에는 리버티게임에서 해당 게임을 표절하는 과정을 거쳐 게임을 만든 척하고, 존재하지 않으면 뺀질거리다가 아무말이나 내뱉습니다. 실제 AI를 사용하지는 않았고 사전에 정의된 메시지로 답합니다.

오른쪽 위에는 옆그레이드 버튼이 요란하게 반짝이며, 옵션은 무료인 [연구소에 요청](https://libertygame.work/wiki/리버티게임:연구소/AI에게_부탁한_게임), 29100원의 Pro, [291000원](https://uncyclopedia.kr/wiki/29만원)의 Infinite Supreme Grand Royal Prime Elite Signature Platinum Diamond Gold Deluxe Premium Advanced Professional Hyper Mega Extreme Super Ultra Pro Max Plus 요금제가 있습니다. 결제를 하려면 그래픽 카드 번호, 유통기한, POV를 작성하고 로봇이 맞습니다에 체크해야 합니다. 그러면 당신은 낚입니다.
